Projekt Stadtrechnungen Basel
"srbas"
=====
This projects collects the XSL/CSS/JS files used in the dissemination package of the Digital Archive of the "Jahrrechnungen Basel im 16. Jahrhundert" project by Susanna Burghartz (Univ. Basel, Departement Geschichte) and Georg Vogeler (Univ. Graz, Zentrum für Informationsmodellierung) <http://gams.uni-graz.at/context:srbas> deployed in the FEDORA-COMMONS based Cultural Heritage Asset Management system GAMS.
2014
